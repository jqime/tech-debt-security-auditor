Términos y Condiciones

Última actualización: 25 de mayo de 2026

1. Servicio

CodeAudit Pro es una plataforma de auditoría de seguridad y cumplimiento normativo. El servicio incluye escaneo automatizado de código fuente, detección de vulnerabilidades, medición de deuda técnica y generación de informes de cumplimiento NIS2/DORA.

2. Responsabilidades del Cliente

  • El cliente declara ser propietario o tener autorización expresa para auditar el código fuente proporcionado.
  • El cliente no debe enviar código malicioso, ilegal o que infrinja derechos de terceros.
  • El cliente es responsable de rotar cualquier secreto o credencial detectado durante la auditoría.

3. Uso de Datos

  • El código fuente se almacena temporalmente durante el proceso de auditoría y se elimina inmediatamente después.
  • Los informes generados se almacenan de forma segura y solo son accesibles por el cliente y el equipo de CodeAudit Pro.
  • No compartimos ni vendemos datos de clientes con terceros.

4. Limitación de Responsabilidad

CodeAudit Pro proporciona herramientas de análisis automatizado. Los resultados no constituyen una auditoría de seguridad formal ni una certificación oficial. El cliente reconoce que ningún escáner automatizado puede detectar el 100% de las vulnerabilidades.

5. Facturación

  • Los pagos se procesan a través de Stripe. No almacenamos datos de tarjetas de crédito.
  • Las suscripciones se renuevan automáticamente salvo cancelación expresa.
  • No se realizan reembolsos parciales por períodos no utilizados.

6. Propiedad Intelectual

El informe generado es propiedad del cliente. El software y la metodología de análisis son propiedad de CodeAudit Pro.

7. Ley Aplicable

Estos términos se rigen por la legislación española. Para cualquier controversia, las partes se someten a los juzgados de Madrid.
